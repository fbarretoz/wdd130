This is an example of a header using CSS Grid layout.
Note that the header structure has three grid items:
  1. img
  2. h1
  3. nav

And uses these CSS grid related declarations

display: grid;
grid-template-columns: 120px 1fr 1fr;
align-items: center;

display: makes the header a grid

grid-tempalte-columns: defines that there are three columns, the first one at a fixed size of 120px and the rest sharing equally the rest of the space

align-items: aligns the grid items to the middle of the header (the grid)
